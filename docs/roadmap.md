# Maggie Roadmap

## Build order

1. Agent loop
2. Tool dispatch
3. Todo tracking
4. Subagent isolation
5. Skill loading
6. Context compaction
7. Persistent task board
8. Background jobs
9. Team messaging
10. Protocols
11. Autonomy

## Architecture principles

- The model decides when to act.
- The harness executes safely.
- State that matters should become files.
- Start with one agent before adding multi-agent behavior.
